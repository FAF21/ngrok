# Install on Debian-based:

* _Puedes clonarlo en el directorio que quieras:_
 
```sh
  git clone https://github.com/tony23x/beef-for-cerberus.git
```

* _Entra a la herramienta:_
```sh
   cd ngrok-install
```
* _Ejecuta el script:_
```sh
   bash ngrok-install.sh
```
